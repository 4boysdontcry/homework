# HTML 정리
  ## div, h1~6, p, div, ol, ul, li : 영역 (block 요소 - layout 요소)
  ## a, img, 글씨 : 영역안에 표현되는 내용 (inline 요소)
  ## class, tag, id 등의 태그의 속성을 attribute 라고 한다.
<br>
<br>

# Emmit 예
```bash
html>head>title{큰제목}^body>h1{제목}+p{내용}
↓
<html>
<head>
  <title>큰제목</title>
</head>
<body>
  <h1>제목</h1>
  <p>내용</p>
</body>
</html>
```

# VScode 단축키
1. alt + shift + 화살표 아래 -- 한줄복사
2. ctrl + / -- 주석 설정해제
3. alt + shift + a -- 선택한 영역만 주석 설정/해제
4. ctrl + , -- 설정
5. ctrl + space -- 자동완성 켜기/끄기
6. ctrl + b -- 탐색기 보이기/끄기(토글, Toggle)

# 해상도
1. dpi - dot per inch (인쇄) ex)300dpi=1인치당 점을 300개 찍음
2. ppi - pixel per inch (화면)
3. 해상도
   1. 720*480 : SD(480p)
   2. 1280*720 : HD(720p) -> 랩탑에서 가장 많이 쓰는 모니터 해상도
   3. 1920*1080 : FHD(1080p) -> 데스크탑에서 가장 많이 쓰는 해상도
   4. 2560*1440 : QHD(1440p) -> 고가형
   5. 3840*2160 : 4K(2160p) -> 현존 최고
4. 디바이스별(스마트폰, 태블릿, 노트북, 데스크탑)해상도
   1. 스마트폰 세로 - 576px
   2. 스마트폰 가로 = 태블릿 세로 - 768px
   3. 태블릿 가로 - 1024px
   4. 노트북 - 1200px
   5. 데스크탑 - 1200px 이상
